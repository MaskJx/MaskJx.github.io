GitHub + Hexo（Next主题）搭建博客

发表于 2018-03-10 | 分类于 技术研究

前言
之前就一直想搭建一个个人技术博客的网站了，但是那时候接触的东西不多，后来看大佬的博客才了解了Hexo，但是一直没有定下决心去看官方文档，于是搭建博客网站的想法再一次不了了之，这次放假之余的闲暇的时候起了搭建博客的想法（windows下），而且之所以写这篇文章是因为自己走了比较多的坑，浪费了比较多的时间，所有把自己搭建的过程写下来，免得自己过后遗忘。

搭建环境

1. 下载并安装 node.js(官网下载)
2. 下载并安装 Git （官网下载）
3. 下载并安装 Hexo。 方法： 打开CMD运行npm install -g hexo

本地搭建Hexo博客

  1. 新建一个文件夹，如MaskJx_Blog（可以在本地随便一个位置，但是最好不要放在C盘）
  2. 进入该文件夹内，右击运行git Bash Here，输入：hexo init（生成Hexo模板）

    稍微等待下，速度有点慢。成功提示：INFO Start blogging with Hexo!

  3. Git运行如下命令
    hexo server（启动本地服务）
    最后在浏览器中输入localhost:4000,可以看到网站初步的模样。

将博客与GitHub关联

  1. 注册GitHub账号，并创建一个名字为XXX.github.io的项目，XXX为自己的GitHub的用户名。（不会的小伙伴可以参考 GitHub 教程）
  2. 完成第一步之后，打开CMD，输入 ssh-keygen -t rsa -C "GitHub的注册邮箱地址"
    一路Enter过来就好，等密钥生成完毕，会得到两个文件id_rsa和id_rsa.pub，参考 ssh key详解

此时，打开浏览器，访问https://maskjx.github.io (可以在互联网上浏览)

绑定域名

  博客已经搭建好，也能通过github的域名访问，如果想要自己的域名，那就需要设置自己的域名绑定到GitHub这个博客项目上。(如果没有域名的小伙伴去注册一个)

域名添加DNS解析
github官网提供两个主机IP地址：192.30.252.153、192.30.252.154，将这两个地址作为主机地址，进入购买购买域名的管理后台，DNS修改。

  1. 添加两条A记录：

    记录类型        主机记录            记录值
A             @             192.30.252.153
A            @            192.30.252.154

  2. 添加一条CNAME记录：
    CNAME -> www -> maskjx.github.io //原本用来访问github博客的地址

  3. 博客添加CNAME文件：
    配置完域名解析后，进入博客目录，在source目录下新建CNAME文件（没有后缀），编辑CNAME文件，添加 maskjx.me(自己的域名)

  4. 配置统一资源定位符
    打开站点配置文件（MaskJx_Blog文件夹下的_config.yml文件）
    如果有个人域名的话可以设置，否则跳过即可。如果是githu的网址，也是可以填上的。
    url: http://maskjx.me/ 其他保持默认。

更换Hexo主题

到此，博客搭建完毕，域名也能正常解析，但是发现默认的主题并不是自己喜欢的，不要急，换一个就好了。

  1. 下载主题
    进入/MaskJx_Blog/themes 右击运行Git
    输入命令：git clone https://github.com/iissnan/hexo-theme-next next
    在themes文件夹下多出一个名为next的主题文件夹（可以重命名）
  2. 打开站点配置文件，找到 theme 属性，将其值改为next（主题文件夹的名字）
    打开Git，运行 `hexo s` 打开浏览器本地浏览可以看到效果。

添加社交链接

在个人博客中加入自己的微博、知乎和Github链接以提高访问量，接下来了解一下社交链接如何添加：
添加链接及图标
在主题配置文件中找到social属性，把social前的#删除，在其下方添加社交链接，格式为：

社交平台名称：链接 || 图标

链接图标全部来自于 Font Awesome

如果没有找到指定的图标，将会启用默认的图标。

注意：enable 必须为true，不然无法使用图标

添加友情链接

在主题配置文件中，找到Blog rolls属性

设置个人头像

在主题配置文件中，找到Sidebar Avatar属性

 先将avatar字段前的#删除，然后再其后写上头像的目录位置或者链接。
 将头像图片保存再主题目录下的 source/images 文件夹，也可以存放在source/upload_images的文件夹(没有自己新建)，也可以将自己的头像图片，保存在百度网盘或者图床，然后将对应的url地址复制过来，添加在avatar字段后即可。

设置代码高亮主题

NexT 使用 Tomorrow Theme 作为代码高亮，共有5款主题供你选择。 NexT 默认使用的是 白色的 normal 主题，可选的值有 normal，night， night blue， night bright， night eighties：
在主题配置文件下，更改highlight_theme属性

# Code Highlight theme
# Available value:
# normal | night | night eighties | night blue | night bright
# https://github.com/chriskempson/tomorrow-theme
highlight_theme: normal

代码块样式自定义

打开\themes\next\source\css_custom\custom.styl,向里面加入：(颜色可以自己定义) 

/ Custom styles.
code {
    color: #ff7600;
    background: #fbf7f8;
    margin: 2px;
}

设置博文内链接为蓝色

链接的默认颜色是白色的，和普通字体颜色相同，不容易区分.
修改方法

\themes\next\source\css_common\components\post\post.styl

打开post.styl文件，在文件内容的底部添加，如下代码：

  .post-body p a{
  color: #0593d3;
  border-bottom: none;
  &:hover {
    color: #0477ab;
    text-decoration: underline;
  }
}
